Navidrome Toolbox

This app runs the nd-toolbox CLI container alongside your Umbrel Navidrome data.

Use Umbrel's terminal or SSH to enter the toolbox container. Start with dry-run operations and back up before changing your library. 
docker exec -it blulyk-nd-toolbox_toolbox_1 sh
make help
make nd.backup
make sh.remove-unsupported
make beet.import
make beet.duplicatez
make nd.load-duplicates
make nd.eval-deletable

Mounted paths: /navidrome for the Navidrome database, /music for Umbrel music downloads, and /data for toolbox data and backups.
